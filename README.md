# Python列表添加元素的三种方法

## 内容概述

1. **使用`append()`方法**  
   详细介绍了如何使用`append()`方法向列表末尾添加单个元素。

   2. **使用`extend()`方法**  
      解释了如何使用`extend()`方法将另一个列表中的所有元素添加到当前列表中。

      3. **使用`insert()`方法**  
         展示了如何使用`insert()`方法在列表的指定位置插入元素。
